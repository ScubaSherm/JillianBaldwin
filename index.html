  1. Which behavior is occurring the most? 
  1. Specified behavior:   
    1. If the specified behavior is occurring most often, genuine assent is likely present. Continue with the program and revisit the tool as necessary.
  2. Alternative behavior:
    1. If the alternative behavior is occurring most often, genuine assent is likely present, but the program needs to be modified in order to help the learner engage in the specified behavior. Move on to Tool 2.
  3. Programmed opt-out response:
    1. Genuine assent is likely not present. The learner is frequently engaging in the opt-out response to escape or avoid the program. The program is likely in need of modifications. Move on to Tool 3.
  4. Other opt-out response:
    1. Genuine assent is likely not present. The learner is frequently engaging in behavior to escape or avoid the program. The program is likely in need of modifications. Move on to Tool 3.
